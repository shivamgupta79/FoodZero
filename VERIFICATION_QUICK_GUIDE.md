# 🏥 NGO Verification - Quick Visual Guide

## 🎯 For NGOs: "How do I get verified?"

### Answer: **Verification happens automatically during registration!**

---

## 📝 Step-by-Step Process

### Step 1: Register as NGO
```
Go to: http://localhost:3000/register

Fill in:
✓ Name
✓ Email  
✓ Password
✓ Select Role: "🏥 NGO"

👇 Additional fields appear automatically 👇
```

### Step 2: Fill NGO Verification Details
```
Mandatory Fields:
✓ Registration Number (e.g., MH/2020/123456)
✓ Registration Type (Trust/Society/Section8)
✓ Registered Address
✓ City
✓ State
✓ Contact Person Name
✓ Contact Phone

Optional Fields:
○ GST Number
○ PAN Number
○ Website
```

### Step 3: Click "Create Account"
```
✅ Account created!
✅ Verification automatically submitted!
✅ Admin automatically notified!

You'll see message:
"NGO registered successfully. Your account is 
pending verification by admin."
```

### Step 4: Login and See Status
```
Login → NGO Dashboard

You'll see YELLOW BANNER:
┌─────────────────────────────────────┐
│ ⏳ Verification Pending             │
│                                     │
│ Your NGO account is under review    │
│ Cannot accept donations yet         │
│                                     │
│ 📋 What happens next:               │
│ • Admin reviews your details        │
│ • Takes 24-48 hours                 │
│ • You'll be notified                │
│ • Then you can accept donations     │
│                                     │
│ ℹ️ How did I apply?                 │
│ Your verification was automatically │
│ submitted when you registered!      │
└─────────────────────────────────────┘
```

### Step 5: Wait for Admin
```
⏰ Typical wait time: 24-48 hours

What you CAN do:
✓ Browse donations
✓ See donation details
✓ View dashboard

What you CANNOT do:
✗ Accept donations (buttons disabled)
✗ Update status
```

### Step 6: Get Verified!
```
When admin approves:

🔔 Notification: "🎉 Your NGO has been verified!"

Login → See GREEN BANNER:
┌─────────────────────────────────────┐
│ ✅ Verified NGO                     │
│                                     │
│ Your NGO is verified and can        │
│ accept food donations               │
└─────────────────────────────────────┘

✅ All buttons now enabled!
✅ Can accept donations!
✅ Full access!
```

Done!

## ❓ FAQ

### Q: Do I need to fill a separate verification form?
**A:** No! Verification details are part of registration.

### Q: When is my verification submitted?
**A:** Automatically when you click "Create Account" during registration.

### Q: How do I know my verification is pending?
**A:** You'll see a yellow banner on your dashboard.

### Q: Can I check verification status?
**A:** Yes! Login and check the banner on your dashboard.

### Q: How long does verification take?
**A:** Typically 24-48 hours.

### Q: Will I be notified when verified?
**A:** Yes! You'll receive a real-time notification.

### Q: What if I'm rejected?
**A:** You'll see the rejection reason on your dashboard. Contact admin to resolve.

### Q: Can I update my verification details?
**A:** Contact admin if you need to update information.
